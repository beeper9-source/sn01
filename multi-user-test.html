📋 테스트 방법

  2. 메인 출석부 열기: 각 탭에서 index.html을 열어주세요
  3. 동시 수정 테스트: 한 탭에서 출석 상태를 변경하고 다른 탭에서 실시간 반영 확인
  4. 네트워크 테스트: 네트워크를 끊었다가 다시 연결하여 오프라인/온라인 동작 확인
